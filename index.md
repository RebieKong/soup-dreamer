# 汤达人菜谱软件

## FUNCTION

     1. 配置症状与诱因的关联
     2. 配置症状的解决方法
     3. 配置解决方法的对应菜单
     4. 配置菜单的食材
     5. 配置食材的信息
     6. 配置工艺的信息
     7. 允许用户交流
     8. 允许分享菜单

## 工作步骤

     0. 数据模型设计
     1. 前端界面设计
     2. 技术选型讨论
     3. 前端开发
     4. 后台开发
    
## 人员招募

    1. 产品经理 * 1
    2. 前端开发 * 1
    3. 后端开发 * 1
    4. 设计人员 * 1
    5. 意见技术提供者 * N
